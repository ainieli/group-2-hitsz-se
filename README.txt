组长：肖安琪
组员：陈筱雯，闫宏书，李昕阳，祝清麟

说明：
文件夹中包含一个star_movie.exe，为本程序的入口，其余文件为相关的支持与配置文件，请勿随意更改
项目原始文件为movie_sys.py server.py idcode.py和main.kv，其中movie_sys.py为主模块
允许在Windows 7及以上的版本运行
注意：Windows 10系统上的搜索功能使用中文时，可能不会显示候选词，但可以盲打或者粘贴将想要的内容输入框中


开发环境：
Python 3.7
实现技术：
使用服务器进行数据管理和爬虫，客户端在联网情况下向服务器发请求获取最新数据，断网情况下使用本地数据。
由于数据量不大，程序一次将所有数据加载到内存中。如果数据有更新，则重新加载更新部分。
使用kivy开发UI，pandas、numpy处理数据，matplotlib绘图，reportlab导出PDF，python开发后台，其中.kv文档存储前端及渲染。


完成的基础功能：
	所有要求的基本功能
完成的额外功能及对已有功能的优化：
	1.对“最佳导演”的可视化（按年份对票房总额进行排序）
	2.同一可视化功能多种形式的图片的选择
	3.部分可视化功能多种排序形式（如劳模演员可按评分、票房、出演次数等）
	4.箱线图分析票房走势，可以直观地看到某些年份中较高的票房与平均值、大部分电影票房数据分布位置之间的差距
	5.搜索功能，可以对主题、电影、演员、导演进行模糊搜索，且可根据票房或评分进行排序
	6.搜索保留历史。在用户（包括游客）一次登录系统注销之前，可以查看到自己曾经搜索过的所有历史信息，单击历史条目即可省略输入的过程
	7.增加随机台词和随机音乐，优化用户体验（可在目录页关闭音乐）
	8.对存储的用户密码与验证问题，加密保存
	9.保存pdf报表的时候，可以用户自己选择路径（默认路径为桌面，单击进入目录）
