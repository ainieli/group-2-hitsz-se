HITSZ 2016级 软件工程 第二组项目 星辰电影
组长：肖安琪
组员：陈筱雯，闫宏书，李昕阳，祝清麟

文件夹中包含一个star_movie.exe，为本程序的入口，其余文件为相关的支持与配置文件，请勿随意更改
项目原始文件为movie_sys.py server.py idcode.py和main.kv，其中movie_sys.py为主模块
允许在Windows 7及以上的版本运行
注意：Windows 10系统上的搜索功能使用中文时，可能不会显示候选词，但可以盲打或者粘贴将想要的内容输入框中


完成的基础功能：
	所有要求的基本功能
完成的额外功能：
	1.对最佳导演的可视化（按年份对票房总额进行排序）
	2.同一可视化功能多种形式的图片的选择
	3.箱线图分析票房走势，可以直观地看到某些年份中较高的票房与平均值、大部分电影票房数据分布位置之间的差距
	4.搜索功能，可以对主题、电影、演员、导演进行模糊搜索，且可根据票房或评分进行排序
	5.搜索保留历史。在用户（包括游客）一次登录系统注销之前，可以查看到自己曾经搜索过的所有历史信息，单击历史条目即可返回搜索的结果
	6.增加随机台词和随机音乐，优化用户体验（可关闭音乐）
	7.对存储的用户密码与验证问题，加密保存


更新：

		1.新增使用MD5算法对用户密码和验证问题进行加密；
